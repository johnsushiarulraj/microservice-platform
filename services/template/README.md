# microservice-template

Generated by [JavaBackend](http://localhost:18090/devconsole/).

## Quick Start

1. Open this folder in IntelliJ or VS Code (Import as Maven project, SDK: Java 11)
2. Edit `provision.yml` if you need additional infrastructure
3. Deploy:

```
deploy.bat 1.0.0
```

This builds, provisions infrastructure, and deploys to the platform in one command.

## Commands

| Command | What it does |
|---------|-------------|
| `deploy.bat 1.0.0` | Build + provision + deploy (one command) |
| `build.bat 1.0.0` | Build JAR + Docker only (no deploy) |
| `test.bat` | Run unit tests |

## URLs (after deploy)

- Health: http://localhost:18090/template/actuator/health
- API: http://localhost:18090/template/api/items
- Docs: http://localhost:18090/template/swagger-ui.html

## Infrastructure

Edit `provision.yml` to add/remove databases, topics, queues, buckets.
Run `deploy.bat` again to apply changes.
